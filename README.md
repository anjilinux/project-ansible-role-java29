ansible-role-java-openjdk
=========

Install java OpenJDK from the official tarball

Role Variables
--------------

```
java_openjdk_url: "https://download.java.net/java/GA/jdk11/9/GPL/openjdk-11.0.2_linux-x64_bin.tar.gz"

java_openjdk_install_folder: "/opt"
```
